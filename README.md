## About
**NexTicket** centralizes tickets from GLPI, Solman, Email & Chatbot into a single intelligent platform. It uses AI/NLP to auto-classify & prioritize tickets, routes to the right technicians, and provides a self-service chatbot + knowledge base, plus dashboards for SLA and productivity.

---

## Highlights
- Unified ingestion from multiple channels  
- AI/NLP microservice for automatic classification & priority prediction  
- Self-service chatbot that can create tickets automatically  
- Real-time dashboard (Recharts) for SLA & productivity metrics  
- Ready for hackathon submission: demo video, screenshots, and deployable code
